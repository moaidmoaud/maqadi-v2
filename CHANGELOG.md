# Sprint 2.5 — المرحلة الثانية

## تحسينات الجودة والأداء
- حفظ تلقائي محسّن مع تقليل عمليات الكتابة المتكررة.
- حماية من تلف البيانات المحلية حتى لا يتوقف التطبيق عند التشغيل.
- بحث عربي أدق: إزالة التشكيل، توحيد الهمزات، ودعم البحث بعدة كلمات.
- ترتيب نتائج البحث حسب التطابق ثم المفضلة وكثرة الاستخدام.
- إنشاء معرفات أكثر أمانًا للعناصر والقوائم.

## تحسينات الاستخدام
- تعديل اسم الغرض وإعادة تصنيفه تلقائيًا.
- التراجع عن حذف الغرض من خلال رسالة سفلية.
- حذف جميع الأغراض المنتهية بعد التأكيد.
- إعادة جميع الأغراض إلى حالة «متبقي» لبدء جولة جديدة.
- تحديث اسم ملف APK في GitHub Actions للمرحلة الثانية.
