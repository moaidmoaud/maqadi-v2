# مقاضي v2 — Sprint 2.5 المرحلة الثالثة

نسخة Flutter أصلية للأندرويد تركز على اللمسات الاحترافية وتجربة الاستخدام.

## الجديد
- صفحة مفضلة فعلية قابلة للإدارة.
- صفحة إحصائيات متجاوبة.
- إعدادات مظهر محفوظة: النظام، الفاتح، الداكن.
- تغيير حجم النص وحفظه.
- حركات خفيفة لشريط التقدم والعناصر المنتهية.
- تحسين التوافق مع الشاشات الكبيرة والصغيرة.
- GitHub Actions جاهز لبناء APK.

## البناء
ارفع محتويات هذا المجلد إلى جذر المستودع، ثم شغّل Workflow باسم Build Android APK.
