# مقاضي v2 — Sprint 2.6 المرحلة الثانية

تطبيق Flutter عربي لإدارة قوائم المقاضي ومخزن المنزل، ويعمل محليًا دون إنترنت.

## البناء من GitHub
1. ارفع محتويات هذا المجلد إلى جذر المستودع.
2. افتح تبويب **Actions**.
3. اختر **Build Android APK** ثم **Run workflow**.
4. حمّل Artifact باسم `Maqadi-Sprint-2.6-Phase-2-APK`.

## أبرز مزايا المرحلة الثانية
- زر **تم وضع المقاضي في المنزل** داخل قائمة التسوق.
- نقل المنتجات المحددة كمشتراة إلى مخزن المنزل وتحديث الكميات تلقائيًا.
- إنشاء منتج جديد في المخزن تلقائيًا إذا لم يكن موجودًا.
- إزالة المنتجات المنقولة من قائمة المقاضي بعد تأكيد العملية.
- سجل حركة لكل منتج: شراء، استهلاك، إضافة وتعديل.
- لوحة مخزن تعرض الإجمالي، الطبيعي، المنخفض والمنتهي.
- إضافة المنتجات المنخفضة والمنتهية إلى قائمة مقاضي نشطة.
- حفظ جميع البيانات وسجل الحركات محليًا.

## ملاحظة الاختبار
هذه نسخة Beta للتجربة. يفضّل الاحتفاظ بنسخة من المرحلة الأولى إلى أن تتأكد من أن سير العمل مناسب لك.
