# مقاضي — Sprint 2.5 المرحلة الثانية

هذه الحزمة تبني فوق المرحلة الأولى وتركز على الاستقرار والسرعة وتحسين البحث وتجربة الاستخدام.

## أبرز التحسينات
- حفظ محلي آمن وأخف على الجهاز.
- بحث عربي محسّن ومتعدد الكلمات.

- تراجع عن الحذف.
- تنظيف الأغراض المنتهية وإعادة القائمة لجولة جديدة.
- GitHub Actions جاهز لبناء APK.

## التثبيت
