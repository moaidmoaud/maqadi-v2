# مقاضي — Sprint 2.5 المرحلة الأولى

هذه الحزمة تطور المشروع من قائمة واحدة إلى نظام قوائم متعددة.

## الموجود في هذه المرحلة
- صفحة رئيسية جديدة.
- إنشاء عدة قوائم.
- فتح آخر قائمة بسرعة.
- إعادة تسمية ونسخ وأرشفة وحذف القوائم.
- شاشة تسوق محسنة مع التقدم وإخفاء المشتريات.
- قاعدة المنتجات والبحث الذكي والمفضلة من Sprint 2.4.
- حفظ محلي بالكامل.
- GitHub Actions لبناء APK.

## طريقة الرفع
ارفع محتويات هذا المجلد إلى جذر مستودع `maqadi-v2` مع استبدال الملفات القديمة، ثم شغل Workflow باسم `Build Android APK`.
